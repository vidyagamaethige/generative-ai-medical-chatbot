# End-to-end-Generative-AI-Medical-Chatbot

This is a Python-based medical chatbot powered by Generative AI, designed to provide reliable medical information, symptom explanations, and health guidance using advanced NLP and retrieval techniques.

## Installation

### Step 1 - Clone the Repository
```bash
git clone <repository-https-link>
```

### Step 2 - Go to the project folder 
```bash
cd <project-folder-name>
```

### Step 3 - Create a conda environment after opening the repository
```bash
conda create -n medibot python=3.10 -y
```
### Step 4 - Activate conda environment
```bash
conda activate medibot
```

### Step 5 - install the requirements
```bash
pip install -r requirements.txt
```
